# 仕様

## 概要

- ここん（kokon）は古今東西ゲームを補助するためのSPAです
- 単語データをspreadsheetから取得し、ユーザーの発話に対して正解・不正解を提示します
- Desktop / Mobile 両方のブラウザに対応

## 採用技術

- React
- Vite
- tailwindcss
- HeroUI (必要に応じて)
- csv-parse（spreadsheetのパース）
- Vitest

## 画面構成

- 「単語リスト」「設定・オプション」「スコア・ポイント」で構成される
- 単語リスト
  - 単語データをテーブルとして表示
  - カテゴリごとテーブルを分ける
  - 初期状態では単語は伏せられており、正解すると表示される
- 設定・オプション
  - 「spreadsheetのURL入力欄」「回答状況のリセットボタン」「画像の表示非表示」などがある
  - 隠せる
- スコア・ポイント
  - プレイヤーの正解数を表示する

## ユースケース1: 回答データ取得・表示

- spreadsheetのURLからデータを取得する
- データは「カテゴリ」「単語」「読み」「画像」から構成される。画像はないこともある
- カテゴリごとテーブルとして表示する（読みは表示しない）
- 初期状態は単語・画像は伏せられている
- 正解した単語・画像は表示する

## ユースケース2: ゲームプレイ

- ユーザーは数字のボタンを押しながら発話する
- ボタンを話したとき、- 発話単語と一致する「読み」があるかチェックする
- 一致する「読み」が複数ある場合は複数正解とする
- 一致する「読み」がない場合は不正解とする
- 正解の場合は正解音を鳴らし、該当する単語・画像を表示する
- 不正解の場合は不正解音を鳴らす
- 正解の場合はボタンに応じた数字のプレイヤーのスコアを1つ増やす（複数正解でも1）
